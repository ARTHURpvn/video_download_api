🎯 O QUE É ESTE APLICATIVO?
───────────────────────────────────────────────────────────────────────────────
Aplicativo desktop para baixar vídeos e áudios do YouTube.
Instalação totalmente automática - o instalador cuida de tudo!


══════════════════════════════════════════════════════════════════════════════
                        🪟 INSTALAÇÃO - WINDOWS
══════════════════════════════════════════════════════════════════════════════

SUPER SIMPLES - APENAS 2 PASSOS:
  1. Duplo clique em: INSTALAR_WINDOWS.bat
  2. Aguarde 10-15 minutos

✅ PRONTO! O executável estará na sua Área de Trabalho!

O instalador faz TUDO automaticamente:
  • Detecta se Python está instalado
  • Instala Python automaticamente se necessário
  • Instala todas as dependências
  • Compila o executável
  • Coloca na sua Área de Trabalho

⚠️ IMPORTANTE: Você NÃO precisa ter Python instalado antes!
   O instalador cuida de tudo sozinho!


══════════════════════════════════════════════════════════════════════════════
                        🍎 INSTALAÇÃO - macOS
══════════════════════════════════════════════════════════════════════════════

SUPER SIMPLES - APENAS 3 PASSOS:
  1. Abra o Terminal
  2. Navegue até a pasta: cd /caminho/para/PythonProject
  3. Execute: bash INSTALAR_MACOS.sh

✅ PRONTO! O aplicativo estará na sua Área de Trabalho!

O instalador faz TUDO automaticamente:
  • Detecta se Python está instalado
  • Instala Python via Homebrew se necessário
  • Instala FFmpeg automaticamente
  • Instala todas as dependências
  • Compila o aplicativo
  • Coloca na sua Área de Trabalho

⚠️ IMPORTANTE: Você NÃO precisa ter Python instalado antes!
   O instalador cuida de tudo sozinho!


══════════════════════════════════════════════════════════════════════════════
                        💻 COMO USAR APÓS INSTALAÇÃO
══════════════════════════════════════════════════════════════════════════════

  1. Vá até sua Área de Trabalho
  2. Windows: Duplo clique em "YouTubeDownloader.exe"
     macOS: Duplo clique em "YouTubeDownloader"
  3. Cole um link do YouTube
  4. Escolha MP4 (vídeo) ou MP3 (áudio)
  5. Clique em "BAIXAR AGORA"
  6. Arquivos salvos em: Downloads/Videos Baixados


══════════════════════════════════════════════════════════════════════════════
                        ⚠️ PROBLEMAS COMUNS
══════════════════════════════════════════════════════════════════════════════

❌ Windows: "Python não é reconhecido" após instalação automática?
   → Reinicie o computador e execute o instalador novamente

❌ macOS: "Não foi possível abrir porque é de um desenvolvedor não identificado"?
   → Sistema > Segurança > Permitir mesmo assim

❌ macOS: Erro de permissão ao executar .sh?
   → Execute: chmod +x INSTALAR_MACOS.sh
   → Depois: bash INSTALAR_MACOS.sh
